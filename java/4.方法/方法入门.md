### 2.方法进阶

参数：进入方法的数据

返回值：从方法中出来的数据

```java
/*
定义方法的标准格式
返回值类型：就是方法最终产生的数据结果是什么类型
参数类型：进入方法的数据是什么类型
参数名称：进入方法体的数据对应的变量名称
ps：多个参数用逗号隔开
return:两个作用，停止当前方法，将后面的返回值还给调用处
返回值：方法执行后产生的数据结果
*/
public static 返回值类型  方法名称(参数类型  参数名称){
    方法体;
    return 返回值;
}

public static int sum(int a, int b){
    int result=a + b;
    return result;
}
```

### 3.调用方法

```java
/*
1.单独调用
2.打印调用
3.赋值调用
*/
public static int sum(int a, int b){
    int result=a + b;
    return result;
}
//单独调用
sum(10,20);
//打印调用
System.out.println(sum(10,20));
//赋值调用：数据类型 数据名称 = 方法名称(参数)
int number = sum(10,20);
```

### 4.方法的注意事项

1.方法应该定义在类当中，不能在方法中定义方法

2.方法定义的前后顺序无所谓

3.方法定义后自己不会执行，必须通过调用才能使用

4.如果方法有返回值，那么必须要写上 ==return 返回值==

5.return后面的返回值数据类型必须和方法的返回值类型相同

6.对于一个void没有返回值的方法，不能写return后面的返回值，只能写return

7.对于方法当中最后一行的return在一些情况下可以省略不写

8.一个方法当中可以有多个return语句（选择），但必须保证同时只有一个会被执行
