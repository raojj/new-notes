### static关键字概述

```java
/*
一旦使用了static关键字，那么这样的内容就不在属于对象自己，而是属于类的，凡是本类的对象，都共享一份
1.如果一个成员变量使用了static关键字，则这个成员变量不再属于对象自己，而是属于类 
2.静态static修饰方法：
一旦使用static修饰了成员方法，成员方法就是属于类而不属于对象自己
可以通过对象名进行调用，也可以通过类名称来调用
obj.methodStatic();准确但不推荐，编译后也会被javac翻译成类名称.方法名
Myclass.methodStatic();准确推荐
所以无论是成员变量还是成员方法，都推荐用类名称.成员名
3.注意事项
静态不能直接访问非静态（就是说静态方法不能访问非静态的成员变量）因为在内存中先有的静态后有的非静态。
静态方法中不能使用this，因为this代表当前对象，通过谁调用的方法，谁就是当前对象
根据类名称访问静态成员变量的时候，全程和对象没有关系，只和类有关系
*/
静态代码块
/*
静态代码块的格式：
public class 类名称{
     static{
     //静态代码块内容（当第一次用到本类时，静态代码块执行唯一的一次，且静态内容有限非静态内容）
     }
}
用途：
用来一次性的对静态成员变量进行赋值
*/
```
